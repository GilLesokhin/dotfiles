# RESOURCES.md Format

`RESOURCES.md` is the curated set of trusted sources for the topic. Draw explanatory knowledge from it, not unverified guesses. Communities provide wisdom.

```md
# {Topic} Resources

## Knowledge

- [Book: _Title_ - Author](https://example.com)
  What it covers and when to use it.

## Wisdom (Communities)

- [Community](https://example.com)
  What it is useful for.
```

- Prefer primary sources, recognized experts, peer-reviewed work, and strongly moderated communities.
- Annotate every entry with what it covers and when to use it.
- Group resources into Knowledge and Wisdom.
- Add a `## Gaps` section when no good source covers a mission-critical area.
- Remove shallow, wrong, or off-mission resources.
- Record a user's community preferences here.
